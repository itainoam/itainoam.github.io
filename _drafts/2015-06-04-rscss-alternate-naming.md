description: A lot of people have adopted [RSCSS] with different naming conventions.

### Single-word components

```
.component
.component > .element
.component.-modifier
```

### PCSS

```
.component
.multiwordcomponent
.component-element
```

Example:

```
.searchbar
.searchbar-textbox
.searchbar-button
.searchbar.-compact
```

[RSCSS]: http://rscss.io
